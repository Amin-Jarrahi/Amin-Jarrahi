## 🧬 About Me

I'm a **PhD Biomedical Engineer** from the **University of Tennessee, Knoxville** (Crouch Imaging Lab), specializing in the intersection of **AI/ML and biomedical data science**. My research bridges spatial omics, mass spectrometry imaging, and computational biology to uncover meaningful biological patterns from complex, high-dimensional data.

- 🔭 **Currently working on:** Enhancing the Multi-Objective Scoring (MOS) framework using **GATv2 (Graph Attention Networks v2)** for improved cross-modal spatial pattern detection in multi-omics data
- 👯 **Looking to collaborate on:** AI/ML for healthcare, spatial omics analysis, biomedical imaging, and multi-modal data integration projects
- 🤝 **Looking for help with:** Scaling ML pipelines for biomedical data, productionizing research tools, and deploying high-performance bioinformatics systems
- 💬 **Ask me about:** MALDI-MSI, spatial transcriptomics, multi-omics integration, graph neural networks, machine learning, causal inference, biomedical imaging, and high-dimensional data analysis
- ⚡ **Fun fact:** I built a C++ system that reduced a two-month biomedical data analysis workflow to just **10 seconds**
